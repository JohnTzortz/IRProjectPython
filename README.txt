In order to run this project you have to install some Libraries.

You have to add the dependencies in IDE pyCharm.
The easiest way to do this is by going to file>settings>Project:NameOfProject and adding them.
For this project, you have to add an interpent, Flask, and Whoosh.

Python version: 3.11

In order to run the application, you run the app.py.
Then open your browser: http://127.0.0.1:5000

It's recommended to refresh the page before you try to use the program.

Firstly, push the green init index button at the bottom.
Wait at least 30-60 seconds for the script to index the dataset. When the indexing is complete,
it will be shown in the Python terminal

When the index is complete, you can use the system.
